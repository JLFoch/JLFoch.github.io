Web Host Research Project

GoDaddy

GoDaddy provides a lot of different resources for their customers. GoDaddy offers domains, websites, website design services, SSL security, Web Hosting, and much more. You can buy a .com for only $12.99 for the first year. GoDaddy offers about 20 different hosting plans that one can get. These plans are grouped across five different types of hosting: shared, WordPress, business, VPS, and dedicated. GoDaddy’s web hosting plans start off at $1.00 per month for the economy plan, $7.99 per month for the Deluxe plan, and $12.99 for the Ultimate plan. GoDaddy does have a very easy to use website builder that people with no website design background can use and be very successful. The website builder costs from $9.99 per month for basic all the way to $24.99 per month for Ecommerce. Website hosting is included in all Website Builder plans, this is a great idea for small businesses. GoDaddy also offers both cPanel Linux and Plesk Windows hosting for the GoDaddy shared hosting platforms. It is also very easy for one to set up eCommerce for all of your big or small business needs. In addition, their help line is open 24/7 which is great for businesses.

Bluehost

Bluehost is a web hosting provider that is ranked on cnet’s list of the best web hosting for 2021. Bluehost pricing seems to be pretty cheap. When you sign up with Bluehost, you already get a free domain name for the first year. In addition, you also get unmetered bandwidth and a free SSL certificate, which is pretty nice. Bluehost is great if you are a WordPress user. Bluehost’s managed WordPress hosting costs a little more than basic shared hosting. Although Bluehost has both specific WordPress and WooCommerce hosting plans available with management support. Bluehost also offers 24/7 support which is great for businesses.

What they offer and pricing.

  • hared hosting starting at $2.95 per month
  • WordPress hosting starting at $ 2.95 per month
  • VPS hosting starting at $18.99 per month
  • Dedicated hosting starting at $79.99 per month

HostGator

HostGator is another a web hosting provider that is ranked on cnet’s list of The best web hosting for 2021. HostGator is a full-service web hosting company. Their plans can fit virtually any need one may have. Shared hosting starting at $2.75 per month, WordPress hosting starting at $5.95 per month, Reseller hosting starting at $19.99 per month, VPS hosting starting at $29.99, Dedicated servers starting at $119 per month, and Cloud hosting starting at $4.95 per month. HostGator only allows more high-level site builders to modify their dedicated or virtual server. They also offer the choice of Linux or Windows hosting. A downside is that it is challenging to figure out what the actual price is once promotional plans run out.

The winner is GoDaddy

The top host I have picked is GoDaddy. There are a few different reasons for this. A huge one is that you do not need any web design background to make a great looking site using GoDaddy’s services. I use their ecommerce website builder because I had no clue how to build a sire from the ground up. The website builder is just so easy to use and you can go so many different things. Although it is limited, they have tons of options. I also like that GoDaddy has 24/7 tech support, this is great for any business. GoDaddy pricing is also pretty good considering you get a lot of different things even with the basic plans. If you have the ability to build a site from the ground up there are plans for that too. There are plans for all skill sets and price ranges.
